THREE-TIER APPLICATION USING DOCKER

Harsh Patel

This is a documentation for developing a three-tier application using docker.

The project is divided into 3 parts as follows : FrontEnd, Backend, Database.

Navigation

  • DATABASE
  • BACKEND
  • FRONTEND

DATABASE TIER

Step 1: Create dockerfile for MongoDB

Step 2: Create Network

List available networks : docker network ls
Create a Docker network : docker network create new-network

Step 3: Connecting MongoDB Container to Network

Build Docker image for MongoDB : docker build -t darshan3112/21bcp129-mongodb ./mongodb

Run MongoDB container : docker run -d --name mongodb-container --network new-network darshan3112/21bcp129-mongodb

BACKEND TIER

Step 4: Create dockerfile for Backend

Step 5: Building and Running Node.js Backend Container

Build Docker image for NodeJS backend : docker build -t darshan3112/21bcp129-backend ./backend

Run NodeJS backend container : docker run -d --name backend-container --network new-network darshan3112/21bcp129-backend

FRONTEND TIER

Step 6: Create dockerfile for Frontend

Step 7: Building and Running React Frontend Container

Build Docker image for Frontend : docker build -t darshan3112/21bcp129-frontend ./frontend

Run frontend container : docker run -d --name 21bcp113-frontend-container --network new-network darshan3112/21bcp129-frontend
